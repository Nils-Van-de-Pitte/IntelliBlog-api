# IntelliBlog CMS – Project Documentation

## Project Overview
IntelliBlog CMS is a modern, lightweight blogging platform aimed at individual content creators and small teams. It's designed for flexibility, scalability, and future AI integration, such as content generation, SEO tools, and image tagging.

---

## Goals & Objectives
- Build a modular CMS focused on blog content.
- Provide an intuitive admin/editor interface.
- Enable API-driven and server-rendered use.
- Plan architecture to easily support AI features.

---

## Target Users
- Independent bloggers
- Content creators and marketers
- Small editorial teams
- Developers wanting an extendable CMS

---

## Core Features (MVP)

### Admin & Editing
- User roles: Admin, Editor
- Secure login/authentication
- Post creation, editing, and deletion
- Media uploads (images/videos)

### Content Management
- Markdown/WYSIWYG editor
- Categories and tags
- Featured image and SEO metadata

### Public Blog Interface
- Homepage with latest/featured posts
- Full post pages with author info
- Comment support (manual or Disqus)

### API Layer
- REST or GraphQL APIs
- JWT token-based authentication
